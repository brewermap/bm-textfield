# bm-textfield



## Installation

This module is installed via npm:

``` bash
$ npm install bm-textfield
```

## Test 

You can run the tests via npm. This executes `web-component-tester`

```
npm test 
```

## Lint

```
npm run lint 
```

## Reset

Resets all local dependencies and deleted package-lock.json

```
npm run reset 
```
